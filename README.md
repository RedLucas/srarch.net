## How to use

- Ensure you have node (18) installed
- run `npm ci`
- run `npx next dev` to deploy locally
- visit http://localhost:3000 to view the current state.

## Stack intentions

- Hosted on Cloudflare pages
- Statically generated by Next.js
- Tailwind css
